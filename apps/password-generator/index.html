Password Generator

Jum'at, 17 September 2021

Membuat password acak dengan mengabungkan kombinasi huruf dan angka.

Aplikasi ini dibuat menggunakan library AlpineJS1 dan password-generator2.

Custom javascript pattern. Contoh: [A-D] menghasilkan kombinasi dari huruf A hingga D saja.

Aplikasi ini tidak menyimpan password yang dibuat. Aktifitas pembuatan password dilakukan dibrowser (peramban) anda tanpa melibatkan sisi server.

  1. https://alpinejs.dev/ ↩︎

  2. https://github.com/bermi/password-generator ↩︎
