Xdebug Mode Off Trigger Makefile Error

Pesan error:

make: *** [Makefile:135: dump-autoload] Error 2816

Solusi:

  1. Hilangkan setting XDEBUG_MODE = "off"
  2. Tambahkan informasi folder composer pada konfigurasi skipped paths di PHPStorm. Lihat tautan ini https://github.com/composer/composer/issues/8302#issuecomment-526908909.
